Purpose

The ff-connect attribute connects a firebase data set to your HTML. ff-connect is required for firefight to work. Only child elements of ff-connect are controlled by firefight. A web page can have only one ff-connect attribute. The <body ff-connect="myAppName"> element is an excellent place for the ff-connect attribute.

options

atr-root allows direct access to child nodes of the data. See Weather Sample for an example.

Sample: simple text

HTML

<div ff-connect="ff-demo">
  {{someValue}}
</div>
